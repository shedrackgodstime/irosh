## [0.1.0] - 2026-04-20

### Initial Release
- **Secure P2P Shell**: SSH sessions over Iroh peer-to-peer transport with no central relay needed.
- **Audit-Proven API**: Fully compliant with Rust API Guidelines (C-CHECKLIST).
- **TOFU Security**: Trust On First Use policy for host keys and client identities.
- **Professional Tooling**: Comprehensive CLI binaries (`irosh`, `irosh-server`, `irosh-client`) for all platforms.
- **Robust Feature Set**: Includes named peer aliases, metadata exchange, and single-file transfer side-streams.
- **Structured Errors**: Granular, typed error system for reliable diagnostics.
- **Production Metadata**: High-quality documentation and professional installation scripts.
